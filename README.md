# ChatBot de Gestión de Pedidos — Telegram

Bot para automatizar la recepción de pedidos mayoristas y registro de proveedores vía Telegram, con integración a Google Sheets en tiempo real.

## Funcionalidades

- Recepción automatizada de pedidos 24/7
- Clasificación de usuarios con IA (cliente / proveedor)
- Validación de datos en tiempo real
- Notificación al equipo con aprobación por botón
- Registro instantáneo en Google Sheets
